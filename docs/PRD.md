# 功德 APP 产品需求文档 (PRD)

## 一、产品概述

### 1.1 产品名称

功德 APP

### 1.2 产品定位

一款面向年轻人段、泛信仰的用户的文化应用，提供虚拟礼佛、功课修行、功德积累、讲经直播、AI 问答等功能。

### 1.3 目标用户

- 面向年轻人段的用户
- 有佛教信仰的用户
- 对传统文化感兴趣的泛用户
- 寻求心灵慰藉的用户

### 1.4 设计风格

- **视觉风格**：传统庄严风格
- **配色**：以金色、红色、深棕色为主色调
- **字体**：使用具有传统韵味的字体
- **图标**：采用佛教元素图标（莲花、法轮、佛珠等）

### 1.5 账户体系

- 支持游客模式，无需注册即可使用
- 本地记录用户数据：功德值、功课记录、礼拜记录、开光记录

## 二、功能模块

### 2.1 礼拜模块 🙏

#### 功能描述

以许愿为主交互，类似打卡形式，用户可选择不同菩萨进行虚拟礼拜。同时提供上香、募捐入口，每日限量一炷香，增加稀缺感和仪式感。

#### 菩萨列表

| 菩萨     | 主管领域   | 场景描述     |
| -------- | ---------- | ------------ |
| 观音菩萨 | 慈悲、平安 | 南海观音殿   |
| 文殊菩萨 | 智慧、学业 | 五台山文殊殿 |
| 地藏菩萨 | 超度、孝道 | 九华山地藏殿 |
| 财神     | 财运、事业 | 财神殿       |

#### 交互流程

```
首页 → 点击"礼拜" → 进入礼拜页
→ 选择菩萨 → 输入许愿内容 → 点击"许愿礼拜"
→ 播放礼拜动画 → 显示"功德+1" → 打卡成功
```

#### 功能规则

- **许愿打卡**：每日可许愿一次，形成连续打卡记录
- **每日限量**：每位用户每日限上 1 炷香
- **功德记录**：每次礼拜成功后功德值 +1
- **礼拜记录**：记录礼拜时间、菩萨、许愿内容

#### 礼拜页元素

- 菩萨形象（2D 插画风格）
- 许愿输入框（核心交互）
- 上香入口按钮
- 募捐入口按钮
- 连续打卡天数展示
- 功德提示动画
- 背景音效：钟声、木鱼声

#### 功德记录页

- 展示所有礼拜记录
- 许愿历史列表
- 功德值统计
- 打卡日历视图

### 2.2 功课模块 📿

#### 功能描述

用户修行为主，提供经书诵读和佛教音乐，通过跟诵音频进行修行，记录功课进度。

#### 功课列表页

分为两个类别：

##### 经书列表

| 经文   | 时长       | 难度 |
| ------ | ---------- | ---- |
| 金刚经 | 约 45 分钟 | 中等 |
| 心经   | 约 5 分钟  | 入门 |
| 大悲咒 | 约 8 分钟  | 入门 |

##### 佛教音乐列表

| 音乐         | 时长    | 类型     |
| ------------ | ------- | -------- |
| 大悲咒音乐版 | 10 分钟 | 诵经音乐 |
| 禅修背景音   | 30 分钟 | 冥想音乐 |
| 钟声梵音     | 15 分钟 | 静心音乐 |

#### 诵经交互流程

```
首页 → 点击"功课" → 功课列表页 → 选择经书/音乐
→ 进入诵经页 → 播放诵经音频 → 显示经文滚动
→ 用户跟诵 → 完成后记录功课
```

#### 诵经页元素

- 经文文本显示（自动滚动，高亮当前位置）
- 音频播放控制（播放/暂停、进度条）
- 诵经计数器
- 功课完成提示
- 修行时长统计

#### 功课记录

- 记录每次功课的经文、时长、完成时间
- 累计功课次数统计
- 修行总时长统计

### 2.3 开光模块 ✨

#### 功能描述

独立的开光模块，支持绑定智能硬件（手串、戒指等）进行开光，提供数字开光证书。

#### 智能硬件绑定

##### 支持的硬件类型

| 硬件     | 说明                   |
| -------- | ---------------------- |
| 智能手串 | 带 NFC/蓝牙的佛珠手串  |
| 智能戒指 | 带芯片的护身戒指       |
| 其他饰品 | 支持扩展的智能饰品     |

##### 绑定流程

```
首页 → 点击"开光" → 绑定智能硬件
→ 扫描/连接硬件 → 确认硬件信息 → 绑定成功
```

#### 开光仪式

##### 线上开光

- 用户提交开光物品信息（照片、名称）
- 选择开光类型（平安、事业、姻缘等）
- 观看开光仪式视频/直播
- 完成开光，生成数字证书

##### 线下开光（智能硬件配合）

- 连接已绑定的智能硬件
- APP 引导开光仪式步骤
- 硬件同步开光状态
- 生成数字开光证书

#### 开光记录页

- 已开光物品列表
- 数字开光证书查看
- 证书详情：物品名称、开光类型、时间、法师（如有）
- 支持证书分享、保存

### 2.4 讲经模块 🎬

#### 功能描述

提供大师讲经直播，用户可实时互动；同时包含线上商城，销售经书、文创、智能硬件等商品。

#### 直播功能

##### 直播列表页

- 正在直播的大师列表
- 预告直播时间表
- 直播回放入口

##### 直播内容

- 大师讲经说法
- 佛学知识分享
- 开光法会直播
- 答疑解惑

##### 互动功能

- 弹幕评论
- 点赞
- 虚拟供养（可与礼拜模块关联）
- 提问互动

##### 直播交互流程

```
首页 → 点击"讲经" → 直播列表页 → 选择直播
→ 进入直播详情页 → 观看直播 → 发送弹幕/点赞
→ 退出直播
```

#### 线上商城

##### 商品分类

| 分类     | 商品举例                               |
| -------- | -------------------------------------- |
| 经书     | 金刚经、心经、地藏经等实体书籍         |
| 文创     | 佛珠、香囊、书签、挂件等               |
| 智能硬件 | 智能手串、智能戒指等（与开光模块联动） |

##### 商城交互流程

```
讲经模块 → 点击"商城" → 商品列表页
→ 选择商品 → 商品详情页 → 加入购物车/立即购买
→ 确认订单 → 支付 → 完成购买
```

##### 商品列表页元素

- 商品分类 Tab（经书、文创、智能硬件）
- 商品卡片（图片、名称、价格）
- 搜索功能
- 筛选排序

##### 商品详情页元素

- 商品图片轮播
- 商品名称、价格
- 商品描述
- 规格选择
- 购买按钮
- 相关推荐

### 2.5 AI 大师对话模块 🧘

#### 功能描述

AI 大师对话功能（Chatbox 形式），回答用户的佛学问题和人生困惑，全局悬浮入口。

#### AI 大师设定

- **人设**：慈悲智慧的虚拟法师
- **语言风格**：温和、有禅意、引经据典
- **回答范围**：
  - 佛学问题（经文解读、修行方法等）
  - 人生困惑（婚姻、事业、健康、情感等）
  - 心灵开导

#### 使用规则

- **每日免费**：10 次对话机会
- **次日重置**：每日 0 点重置免费次数
- **对话记录**：保存历史对话

#### 交互流程

```
任意页面 → 点击"AI 大师"悬浮按钮 → 打开 Chatbox 对话框
→ 输入问题 → AI 回复 → 继续对话或关闭
```

#### 页面元素（Chatbox 形式）

- 悬浮入口按钮（全局可见）
- AI 大师头像
- 对话气泡列表
- 输入框 + 发送按钮
- 剩余次数提示
- 历史对话记录
- 关闭/最小化按钮

## 三、页面结构
